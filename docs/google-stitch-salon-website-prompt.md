## VISUAL DESIGN SPECIFICATIONS

### Color Palette
Use these EXACT colors. Do not substitute or "improve" them.
```
Primary background:    #FAFAF8  (warm cream, never pure white)
Primary text:          #2D2926  (deep charcoal, never pure black)
Accent (gold):         #C9A96E  (muted champagne, used sparingly)
Highlight (blush):     #E8DDD0  (soft blush, for section dividers)
Subtle border:         #E5E1D8  (warm light gray)
Muted text:            #8B8680  (warm gray for captions)
```
NO purple, NO blue, NO neon, NO gradients (except very subtle warm-tone shadows).

### Typography
**Headlines:** Premium serif font. Pick ONE of: Canela, Tiempos Headline, Playfair Display, Cormorant Garamond, GT Super Display. Large sizes: H1 around 64-80px desktop, 40-48px mobile.

**Body:** Clean modern sans-serif. Pick ONE of: Inter, Söhne, Neue Haas Grotesk, Suisse Int'l. Sizes: 16-18px body, 14px small. Line-height generous: 1.6-1.8.

**Hierarchy:** Two weights only — 400 regular, 500/600 medium. Never use 700/800 black weights. Never ALL CAPS except for tiny labels.

### Layout Principles
- Mobile-first but desktop must feel like flipping through a premium print magazine
- Generous whitespace — let content breathe
- Asymmetric grids — not boring centered layouts
- Use overlap and layering for depth
- Grid-breaking elements that create visual interest
- Maximum content width: 1280px, but hero sections can be full-bleed
- Section padding: 120px+ vertical on desktop, 64px on mobile

### Imagery Style
- Editorial beauty photography
- Close-ups of manicured hands, soft-focus portraits, minimalist product shots, ambient salon interiors with warm natural light
- NO stock photo clichés (no women laughing with salads)
- NO fake smiles
- All images should feel like they could appear in Vogue or Harper's Bazaar
- Use warm-toned, natural-light photography
- For placeholders, use Unsplash with these search terms: "beauty salon interior", "manicure close up", "facial treatment", "nail art editorial"

### Motion & Interaction
- Subtle and refined — never aggressive
- Gentle fade-ins on scroll
- Smooth hover transitions (300-400ms ease-out)
- One well-orchestrated hero animation on page load
- NO bouncing elements
- NO parallax scrolling overload
- NO auto-playing carousels

## TECHNICAL REQUIREMENTS

- **Framework:** Generate as React/Next.js compatible code OR clean HTML/CSS that can be easily ported to Next.js 15 App Router
- **Styling:** Tailwind CSS preferred (uses utility classes), with CSS variables for colors
- **Responsive:** Mobile (375px), tablet (768px), desktop (1280px+) — all three perfect
- **Performance:** Lazy load images below the fold, optimize fonts (font-display: swap)
- **Accessibility:** Semantic HTML, alt texts on all images, WCAG AA contrast ratios, focus states on all interactive elements
- **SEO:** Proper heading hierarchy (one H1 per page), meta tags, Open Graph tags
- **Language:** Primary German (de-AT), all content in German
